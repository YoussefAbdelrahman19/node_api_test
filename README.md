# Node API Test

## Description
A Node.js API for serving a SPA and managing blog posts and tags. This project includes functionality to create, read, update, and delete blog posts.

## Features
- CRUD operations for blog posts.



## Technology Stack
- Node.js
- Express
- Morgan for logging
- Jest for testing
